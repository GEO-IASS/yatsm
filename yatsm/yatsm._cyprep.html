yatsm._cyprep module¶

Assorted preprocessing utilities written in Cython for performance

yatsm._cyprep.get_valid_mask¶

Return mask for values of array within band by band min and max ranges

Benchmark showed ~10x speed increase over pure NumPy solution (based on using np.all over multiple min < band < max logics).

Parameters:
  • array (np.ndarray) – 2D array (# band x # observations, np.int32) to mask
  • mins (np.ndarray) – 1D array (# band, np.int32) of minimum range bound
  • maxes (np.ndarray) – 1D array (# band, np.int32) of maximum range bound
Returns:

1D (# observations, np.uint8) array mask with 1’s for observations falling within the range and 0’s for observations outside the specified range
Return type:

np.ndarray
