class yatsm.ts_driver.timeseries_ccdc.CCDCBinaryReader(filename, fformat, dt, size, n_band)[source]¶

Bases: object

This class defines the methods for reading pixel values from a raster dataset. I’ve coded this up because certain file formats are more efficiently accessed via fopen than via GDAL (i.e. BIP).

http://osdir.com/ml/gdal-development-gis-osgeo/2007-04/msg00345.html

Args: filename filename of the raster to read from fformat file format of the raster dt numpy datatype size list of [nrow, ncol] n_band number of bands in image
